# 🎬 YouTube Videos Listing UI

A modern **YouTube-style video listing interface** built using React (TSX) and Tailwind CSS.
This project fetches real video data from an API and displays it in a clean, responsive grid layout.

## ✨ Features

* 🎥 Video cards with thumbnail, title, channel & views
* 📱 Fully responsive grid layout
* 🌙 Dark mode UI (YouTube inspired)
* ⚡ Fast API integration
* 🧩 Component-based architecture
* 🔍 Search bar UI (static)

## 🛠️ Tech Stack

* ⚛️ React (TSX)
* 🎨 Tailwind CSS
* 🌐 REST API
* 📦 Vite
